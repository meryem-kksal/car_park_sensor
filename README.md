# car_park_sensor
Ultrasonik mesafe sensörü kullanarak karşındaki engelle olan mesafeyi santimetre cinsinden ölçtüm. Mesafe kısaldıkça buzzer'ın ötme sıklığını ve LED'in yanıp sönme hızını artırarak gerçek bir araç park sensörü simülasyonu yaptım.
Kullanılanlar: Arduino UNO, HC-SR04 Ultrasonik Mesafe Sensörü, Buzzer, LED, Direnç.
